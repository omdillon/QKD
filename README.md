Hello world!
Welcome to the QKD Simulation Framework. I developed this system over the course of my final year, in fulfillment of my dissertation project.
Both my report and the system are located within the repo, with the bulk of the project written in Python, and part of the experimental modelling completed in MATLAB. To run the system, install requirements.txt, then execute the bash commands found in the simruns.txt under the configs directory. 


This is the culmination of months of work, let me know if there are any issues with installs.
